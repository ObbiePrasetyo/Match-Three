# Match-Three
Match Three Puzzle game

# Getting Started
1. Clone / Fork this repo
2. Give this project Star!
3. Open it with Unity (2018.3.3f1)
4. Enjoy!

# How to Install / Play
1. Go to Build Folder
2. Click Match Three .exe
3. Windowed Screen for Recommend
4. Enjoy!
